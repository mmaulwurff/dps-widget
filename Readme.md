# dps-widget

![screenshot](screenshots/screenshot.png)

dps-widget is an add-on for GZDoom that displays DPS (damage per second).

## Features

- DPS display
- 1-minute DPS history
- DPS graph
- maximal DPS
- average DPS
- total DPS
- customizable looks
- invulnerable target (bind keys)
